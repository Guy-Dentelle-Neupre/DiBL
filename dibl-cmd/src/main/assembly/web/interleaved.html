Commands

A serverside script should execute the command to create the configured diagram. For now you have to install DiBL and copy-paste the desired command into a .sh respective .bat file to execute.

  • sh (Mac / Unix, the multiline string requires quotes)
     
  • bat (Windows, the multiline string requires empty lines)
     

Configure a diagram

pattern  dimensions: nr: 1 max: 7
Once the slider is selected, you can use the arrow keys for the next/previous pattern.
diagram type  thread diagrams not yet implemented.
image type  See edit hints for svg images.
stitches 
Depending on the chosen pattern dimensions, the last rows and columns are ignored.
